# 1. 새로운 데이터인 new_fish의 데이터 세개를 해당 knn모델은 각각 순서대로 도미, 빙어, 빙어로 예측 했다. 그래프 상의 데이터를 육안으로 봤을 때 첫번째 데이터가 도미이고 마지막 데이터가 빙어인 것은 맞아보였으나, 두번째 데이터를 빙어로 분류하는 것은 오류가 있다는 생각이 들었다. 두번째 데이터는 물론 이전의 도미 데이터들 보다 크기와 길이가 가장 작긴 하였지만, 빙어 데이터보다는 도미 데이터의 경향에 더 일치해 보였기 때문이다. 내가 눈으로 확인한 결과와 knn분류기의 예측 결과는 약간 빗나간 것 처럼 보였다.
